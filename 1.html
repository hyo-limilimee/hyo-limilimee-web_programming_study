HTML

CSS HTML[Hypertext Markup Language] 는 인터넷 서비스의 하나인 월드 와이드 웹을 통해 볼 수 있 문서를 만들 때 사용하는웹 언어의 한 종류이다. 특히 하이퍼 텍스트를 작성하기 위해 개발되었으며, 인터넷에서 웹을 통해 접근되는 대부분의 웹 페이지들은 HTML로 작성된다.

1. 태그하기
1) 진하게 표시하기
강조하고 싶은 부분에 강조부분 으로 강조 부분을 감싸주면 됨.

2) 밑줄 긋기
밑줄긋고 싶은 부분에 밑줄 긋고 싶은 부분 으로 밑줄치고 싶은 부분을 감싸주면 됨.

3) 줄 바꾸기
< b r >을 사용한다. 감싸야하는 컨텐츠가 없개 때문에 태그를 닫지 않음.
*닫지 않는 태그
: img, input, br, hr, meta 등


4) 단락을 그룹핑하기
p태그를 사용한다. br과 다르게 열고 닫는 태그가 존재함.*단점: 단락과 단락의 간격이 고정되어 있기 때문에 시각적으로 자유도가 떨어짐.*해결방법: CSS를 사용한다. (HTML : 정보 표현, CSS 정보를 꾸며줌)->

태그를 넣어주면 p 태그의 위쪽에 45px 만큼의 여백이 생김.

5)이미지 넣기
img 태그를 사용함.
